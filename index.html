📖 Comment jouer

🎯 Le but : démasquer l'undercover, le joueur qui n'a pas vu la même vidéo que les autres.

1. L'app distribue les rôles. Tous les joueurs voient la même vidéo… sauf un (l'undercover) qui en voit une légèrement différente.

2. Le téléphone passe de main en main. Chacun regarde sa vidéo en secret.

3. À l'oral, chacun décrit sa vidéo à tour de rôle. Sois assez précis pour ne pas être soupçonné, mais assez vague pour ne pas griller l'undercover si c'est toi.

4. Vous votez pour démasquer l'undercover.

✅ Si le vote est bon → les civils gagnent

❌ Si le vote est mauvais → ce joueur est éliminé, on recommence un tour

🕵️ Si tous les civils sont éliminés → l'undercover gagne
